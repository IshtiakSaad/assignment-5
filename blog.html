Important FAQs

What is the difference between id and class attributes in HTML?

The id attribute is unique and can only be used once per page. It is used to identify a single element uniquely. The class attribute, on the other hand, can be used multiple times and is used to apply the same style or functionality to multiple elements.

How does Flexbox differ from CSS Grid?

Flexbox is a one-dimensional layout model, meaning it works either in a row or a column. It is best used for aligning items in a single direction.

CSS Grid is a two-dimensional layout model, allowing you to work with both rows and columns simultaneously. It's ideal for creating complex layouts with rows and columns.

What is the purpose of the box-sizing property in CSS?

The box-sizing property controls how the total width and height of an element are calculated. By default, the width and height are calculated without including padding and border (content-box), but with box-sizing: border-box, the padding and border are included in the total width and height, making it easier to manage element sizes.

What is the role of event delegation in JavaScript?

Event delegation is a technique that leverages event propagation to handle events efficiently. Instead of attaching event listeners to multiple child elements, you attach a single listener to a parent element. This listener can manage events for multiple child elements, resulting in better performance and easier maintenance.
